Overview and Installation

DDN uses PowerShell to automate many aspects of our job. These tools rely on specific functions that have been created by the team.

In order to access these, you will first need to add the DDN PowerShell Profile to your personal PowerShell instance.

To set up the profile, Navigate to "Tools > Resources > Configure" and run the file called "Install DDN Profile"

Tools

Now that you have set up the profile, you are ready to learn how to use it.

Many of the functions have been built into more robust applications that are stored in our Tools folder.

Instead of having to constantly navigate to this folder in order to use one of the available tools, you can instead enter this one-word command into a PowerShell console:

tools

This command opens a dialog box with a list of all of the available apps; and you will be able to simply click on the one you need in order to run it.

While the most frequently used functions have been converted into more robust applications, there are still many other helpful functions built into the DDN profile.

As you continue to become aquainted with these, another helpful command you can enter into the PowerShell console is:

Get-Comdlets

This command generates a list of all of the custom functions as well as the parameters needed to be able to run them.

While there are many useful tools, the three most commonly used on a daily basis are: Get Apps, Old Swap, and New Swap.

Get Apps

Although Get Apps is available as a robust application, behind the scenes, it is essentially the combination of six custom funcions:

  • Get-ComputerApps -ComputerName EXAMPLE_NAME
  • Get-Serial -ComputerName EXAMPLE_NAME
  • Get-Warranty -SerialNumber EXAMPLE_SERIAL
  • Get-Peripherals -ComputerName EXAMPLE_NAME
  • Get-DHCPLocation -ComputerName EXAMPLE_NAME
  • Get-OU -ComputerName EXAMPLE_Name

It essentially serves as an all-in-one information gathering app.

Old Swap

This function is called by entering:

oldswap

It backs up all of the users files, renames the computer, and shuts it down.

New Swap

This application is found in "Tools>PowerShell" and should only be run locally from the new computer.

It restores the users files, connects their printers, renames the computer, and restarts it.
